使用帮助

1. 首次启动

1. 双击 AListBore.exe 启动应用

2. 等待服务启动完成（约 5-10 秒）

3. 自动打开本地管理页面 http://localhost:5244

4. 使用界面显示的初始密码登录

2. 添加存储

登录后，点击左侧菜单栏的 存储 → 添加

支持的存储类型：

  • 本地存储
  • OneDrive / Google Drive
  • 阿里云盘 / 百度网盘
  • WebDAV / SFTP
  • 以及更多...

3. 外网访问

应用会自动启动 bore 内网穿透，生成外网访问地址

格式：http://bore.pub:端口号

注意：bore.pub 是公共服务，适合临时分享，不建议用于长期存储敏感数据

4. 安全建议

1. 首次登录后立即修改密码

2. 定期备份重要数据

3. 不要在公共网络环境下使用默认密码

4. 如需长期稳定访问，建议使用自己的服务器搭建 bore 服务

5. 常见问题

Q: 启动失败？

A: 请确保 alist.exe 和 bore.exe 与应用程序放在同一目录

Q: 外网地址无法访问？

A: 检查本地防火墙是否允许 5244 端口，或尝试重启应用

Q: 密码丢失？

A: 密码已保存在本地，重新启动应用会自动加载
